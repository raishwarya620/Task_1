# 🌡️ Temperature Converter

A simple and user-friendly *Temperature Converter* application built with *Python* and *Tkinter. This desktop application allows users to convert temperature values between **Celsius, **Fahrenheit, and **Kelvin* using an intuitive graphical user interface (GUI).

---

## 📌 Features

- Convert temperatures between:
  - Celsius (°C)
  - Fahrenheit (°F)
  - Kelvin (K)
- Clean and easy-to-use GUI
- Real-time temperature conversion
- Displays results up to two decimal places
- Input validation with error messages for invalid values

---

## 🛠️ Technologies Used

- *Python 3*
- *Tkinter* (GUI Library)
- *ttk* (Combobox Widgets)
